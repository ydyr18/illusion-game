# 🎴 אשליה - משחק קלפים מיסטי

משחק קלפים מרתק מבוסס JavaScript vanilla עם מולטיפלייר בזמן אמת!

## ✨ תכונות

- 🎮 **מולטיפלייר בזמן אמת** - שחקו עם חברים מכל מקום
- 🎨 **עיצוב מיסטי מהמם** - נושא פנטזיה וקוסמי
- 🃏 **קלפים מצויירים** - כל קלף הוא יצירת אמנות ייחודית
- 📱 **רספונסיבי מלא** - עובד מעולה גם במובייל
- 🔄 **גלילה אופקית חכמה** - תמיכה במשחק ממושך
- 🎯 **מערכת נקודות מתקדמת** - בחירת יעד נקודות

## 🚀 טכנולוגיות

- **Frontend:** Vanilla JavaScript, HTML5, CSS3
- **Backend:** Supabase (Real-time Database)
- **Deployment:** Vercel
- **Avatars:** DiceBear API

## 🎲 איך לשחק?

1. צור משחק חדש או הצטרף למשחק קיים
2. בחר שם ואווטר
3. המארח קובע יעד נקודות
4. גרור קלפים לפי סדר אחוזי הצבעים
5. ערער על רצף לא נכון!

## 📦 התקנה מקומית

```bash
# פשוט פתח את index.html בדפדפן!
open index.html
```

## 🌐 דפלוי

הפרויקט מוכן לדפלוי ל-Vercel:

```bash
vercel deploy
```

## 🔐 הגדרת Supabase

עדכן את ה-credentials ב-`game.js`:

```javascript
const SUPABASE_URL = "YOUR_SUPABASE_URL";
const SUPABASE_ANON_KEY = "YOUR_SUPABASE_KEY";
```
